# Node Express MongoDB RESTful API

### 1. Install MongoDB

https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/

### 2. Run MongoDB
```
mongod
```

### 3. Run Mongoose
```
npm install --save mongoose
```

### 4. Dev Dependency
```
npm install --save-dev babel-cli babel-preset-es2015 mocha rimraf
```
